Recette de crêpes

Voici les différentes étapes pour préparer des crêpes.

Étape 1 : Mettre la farine dans un saladier.

Étape 2 : Ajouter les œufs.

Étape 3 : Mélanger doucement.

Étape 4 : Ajouter le lait petit à petit.

Étape 5 : Mélanger jusqu’à obtenir une pâte lisse.

Étape 6 : Faire cuire les crêpes dans une poêle bien chaude.
