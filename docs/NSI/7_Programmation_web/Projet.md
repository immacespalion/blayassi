# Projet

Vous devez modifier le mini site web en téléchargement ci-dessous ou visible en suivant ce [lien](../../files/NSI/HTML/Projet_HTML_CSS/index.html).

**Toutes les consignes, nombreuses, sont listés sur le lien ci-dessus.**

Vous devez modifier le code HTML et le code CSS, **laissez parler votre imagination** !

Il vous faudra toutefois respecter les contraintes et consignes imposées. Vous serez évalué sur les critères suivants :

- **Respect des contraintes imposées** : qualité du code, liens fonctionnels, utilisation de display:grid, menu...
- **Contenu des pages** : page histoire de l'informatique, sources....
- Travail sur le code HTML, CSS : **qualité** du rendu, **lisibilité**, valide W3C...
